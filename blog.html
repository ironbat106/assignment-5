What is the Document Object Model (DOM)?

The DOM is a programming interface that represents HTML documents as a structured tree.

How do you select an element from DOM?

You can select an element from the DOM using methods like `getElementById`, `querySelector`, or `getElementsByClassName`.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation allows handling events at a parent level rather than individual child elements, improving performance and simplifying dynamic element management.
